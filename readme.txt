A plugin to display the theme name and version number in the admin toolbar area.

== Description ==

This plugin will display the currently activated theme name and version number in the admin toolbar. It is helpful for developers who are often switching between similar themes or versions of themes. 

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress

== Frequently Asked Questions ==

= Where are the plugin options? =

There are not any additional options or configuration - the plugin does not have a settings panel.
